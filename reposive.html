Media Queries

CSS technique that applies different styles based on device characteristics like screen width, resolution, or orientation.

/* Mobile-first base styles */ body { font-size: 14px; } /* Tablet styles */ @media (min-width: 768px) { body { font-size: 16px; } } /* Desktop styles */ @media (min-width: 1024px) { body { font-size: 18px; } }

Fluid Layouts

Using relative units like percentages instead of fixed units to create layouts that adapt to different screen sizes.

.container { width: 90%; /* Fluid width */ max-width: 1200px; /* Max width for large screens */ margin: 0 auto; /* Centered */ } .column { width: 100%; /* Full width on mobile */ } @media (min-width: 768px) { .column { width: 48%; /* Two columns on tablet */ float: left; margin-right: 4%; } }

Mobile-First

Design approach that starts with mobile devices as the primary design target, then enhances for larger screens.

/* Base styles for mobile */ .nav { display: none; } .menu-icon { display: block; } /* Tablet and desktop */ @media (min-width: 768px) { .nav { display: flex; } .menu-icon { display: none; } /* Additional enhancements */ .features { display: grid; grid-template-columns: 1fr 1fr; } }

Flexible Images

Ensuring images resize proportionally within their containers without causing horizontal scrolling.

img { max-width: 100%; /* Never exceed container width */ height: auto; /* Maintain aspect ratio */ } /* For background images */ .hero { background-image: url('hero.jpg'); background-size: cover; /* Cover container */ background-position: center; }

Breakpoints

Specific screen widths where the layout changes to adapt to different device sizes.

/* Common breakpoints */ /* Small devices (phones) */ @media (max-width: 767px) { ... } /* Medium devices (tablets) */ @media (min-width: 768px) and (max-width: 1023px) { ... } /* Large devices (desktops) */ @media (min-width: 1024px) { ... } /* Extra large devices */ @media (min-width: 1200px) { ... }

Responsive Typography

Scaling font sizes and spacing based on screen dimensions for optimal readability.

/* Fluid typography */ html { font-size: 16px; } @media (min-width: 768px) { html { font-size: 18px; } } h1 { font-size: 2rem; /* Relative to base */ line-height: 1.3; margin-bottom: 1.5rem; } /* Using viewport units */ .subtitle { font-size: clamp(1rem, 2vw, 1.2rem); }
